# Perfume Deals Bot

Bot inteligente para detectar ofertas de perfumes en Mercado Libre y publicarlas automáticamente en Telegram.

## Objetivos

- Detectar ofertas reales de perfumes.
- Filtrar únicamente vendedores confiables.
- Mantener historial de precios.
- Calcular un score para cada oferta.
- Publicar automáticamente en Telegram.

## Tecnologías

- Python 3.13
- Playwright
- Selectolax
- PostgreSQL
- FastAPI
- Loguru
- Telegram Bot API
